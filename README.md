# Number Guessing Game

An interactive number guessing game built using HTML, CSS, and JavaScript.  
The application generates a random number and lets the user guess it within a defined range, providing feedback after each attempt.

## Features
- Random number generation
- User guess input
- Feedback hints (too high / too low / correct)
- Attempt tracking
- Win and retry flow

## Tech Stack
- HTML
- CSS
- JavaScript
